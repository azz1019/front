상위폴더로 돌아가기

Regular Expression : 정규표현식, 줄여서 정규식이라고 칭함, RegEx라는 단어로 많이표현됨

  • 특정 데이터의 탐색, 확인, 존재여부등을 쉽게 수행하기위해 만든규칙
  • 정규식이 작동되는 전용메서드에 규칙적인 부호를 사용
  • 일반적인 웹분야에서는 사용률이 낮지만 데이터 분석과 관련된 분야는 마스터를 해야하는 기술
  • 알아두면 복잡한 연산을 쉽게할수 있어서 시간날때 학습하길 권장함
  • 프로그래밍 언어별로 조금씩 차이가있지만 거의 비슷하므로 한번 학습해놓으면 유용하게 사용할수있음

Expressions

  1. str_data.match(/찾는문자/Modifiers) : 찾는문자와 일치하는 값을탐색후 배열리턴
  2. str_data.match(/[찾는문자]/Modifiers) : []안에 찾고자하는 알파벳 혹은 숫자를 넣을수있다, 알파벳이나 단일수로 탐색
  3. [0-9]는 0부터 9까지 해당하는 숫자 : -(하이픈)은 ?부터 ?까지 범위를 지정할때 사용하는 기호, 즉 [0-9]는 모든숫자를 의미함, 문자도 사용가능
  4. [a-zA-Z0-9] : 영문자와 숫자만 가능
  5. str_data.match(/(str1|str2)/Modifiers) : str1이나 str2에 정확하게 부합하는 데이터탐색

MetaCharacter

  1. 브라켓을 사용하는 표현식을 단순하게 특정 문자로 대체한개념
  2. \d : 숫자만, \w : 문자만, etc...
  3. 자주사용되는 메타캐릭터들은 기억해주면 좋다! 매우 유용!

Quantifier : 수량결정자, 수량제한자, 수치표현차

  • n+ : n을 포함하고 n뒤로 최소 1개 이상의문자
  • n* : n을 포함하고 n뒤로 0개 이상의문자
  • n? : n을 포함하고 n뒤로 1개의 문자
  • \d{3} : 3자리 digit character
  • \d{3, 4} : 3 or 4자리 digit character
  • \d{3,} : 3자리이상 digit character
  • ^word : word로 시작하는단어, []안에서 사용하면 not의 의미가됨
  • word$ : word로 끝나는단어
  • word(?= str) : word뒤에 1개이상 str이 오는 word를 리턴
  • word(?! str) : word뒤에 1개이상 str이 오지않는 word를 리턴

RegExp Object Methods

  1. pattern.test('str_value') : 해당패턴이 맞는지여부, 리턴은 boolean
  2. pattern.exec('str_value') : 해당패턴을 값에 대입하여 실행, 리턴은 객체(match메서드와 같음), 객체의 프로퍼티에 접근하면 각 결과값을 알수있음
