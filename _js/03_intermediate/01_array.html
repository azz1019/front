상위폴더로 돌아가기

자바스크립트의 배열은 유동배열 : 크기와 데이터가 변경이 가능하다

  • const my_favors = ['a', 'b', 'c', 1, 2, 3];
  • 원래데이터타입이 같아야하지만 자바스크립트는 허용하고있다
  • my_favors = []; 초기값이 없는 빈 배열을 만든후 데이터를 추가하여 크기변경이 가능하다
  • 각 데이터들은 콤마(,)로 분리되고 분리된자리는 index로 관리되며 01부터 시작
  • 크기는 length(길이)라는 속성을 사용하여 측정하고 크기가 변경될때마다 자동으로 산정된다
  • 인덱스번호 체계에서 순서의 뛰어넘음등의 변칙적인 사용은모두 undefined로 초기화
  • 객체와달리 index로 관리되고 다양한데이터를 저장할수있기 때문에 자바스크립트 프로그래밍에서 활용도가 매우 높으며 이해도가 낮으면 추후 데이터통신 및 데이터변환에서 프로그래밍이 힘들수있다
