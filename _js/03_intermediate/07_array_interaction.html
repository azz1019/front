상위폴더로 돌아가기

배열의 순환할수있는 구조를 이용하여 특정기능을 할수있는 메서드가 제공됨

  • array.reduce(callback function(total, value, index, self), [init value]) : 내부연산의 누적값을 리턴
  • array.reduceRight(reduce메서드와 동일) : 배열의값을 우측에서 좌측방향으로 대입
  • array.every(callback function(value, index, self)) : 콜백함수의 리턴조건에 모든배열의값이 만족하면 true리턴, 그렇지않으면 false리턴
  • array.some(callback function(value, index, self)) : 콜백함수의 리턴조건에 만족하는 배열값이 존재하면 true리턴, 그렇지않으면 false리턴
  • array.indexOf("str value", [start index]) : 배열의 값중 해당하는 문자열이있는 배열의 index값을 리턴함, 문자의 indexOf()메서드와 비슷
  • array.lastindexOf("str value", [start index]) : 배열의 값중 해당하는 문자열이있는 배열의 마지막 index값을 리턴함, 문자의 indexOf()메서드와 비슷
  • array.find(callback function(value, index, self)) : 콜백함수의 리턴조건에 만족하는 첫번째 배열원소를 리턴함, 검색조건 및 값리턴이 쉬운편이여서 자주사용함
  • array.findIndex(callback function(value, index, self)) : 콜백함수의 리턴조건에 만족하는 첫번째 배열원소의 index를 리턴함, 검색조건 및 값리턴이 쉬운편이여서 자주사용함
  • array.from('sequence architecture가 적용된객체') : 순서가있거나 순환구조를 갖고있는 객체를 배열로변환함
  • array.keys() : 배열이 사용중인 key(식별자, 즉 index)를 검색하여 새로운 배열로리턴, 객체에는 적용이안됨
  • array.entries() : 배열의 index:value의 pair구조의 값을 만들어 새로운 배열에 담아리턴, 객체에는 적용이안됨
  • array.includes("찾고자하는값") : 배열에값이 존재하면 true리턴
