상위폴더로 돌아가기

이벤트객체는 전역객체로써 모든 태그에 객체를 참조시킬수 있다!

  • a태그와 button태그에 이벤트를 주로 참조시켜 클릭에대한 반응을 기대할수있는 구조로 만드는게 일반적
  • a태그는 href속성에 의해 자체이동(동기화방식)이 구현되어서 별도의 이동속성을막는 코드가 필요하다
  • 이벤트객체를 부여하는 방식에는 크게 3가지가 있는데 각자 목적과 작동방식이 다르다

이벤트 연습을위해 콘솔이아닌 html문서(document영역)에 출력하는 방식을 이용할예정

  • 화면출력 영역을 document영역(객체)라고 한다
  • html태그는 html Element, 속성은 attribute, 일반텍스트는 textNode라고 함
  • html로 출력하기 : innerHTML이라는 property를 사용
  • text로 출력하기 : innerText라는 property를 사용
  • html element를 id값으로 객체 가져오기 : getElementById('idname')
  • html element를 css선택자를 사용하여 가져오기 : querySelector('selector')
