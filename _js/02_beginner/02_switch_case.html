switch case절 : equal연산을 기반으로 만들어지는 매핑방식의 조건문

equal연산 및 반복 탐색연산에서 if else문보다 속도가 조금더 빠르다고함

여러분은 HTML, CSS, Javascript 과목에대한 테스트를 치뤘다고 가정합시다

  • 각 과목의 점수를 랜덤으로 60~100점까지 생성하여 할당한 후 과목의 총점과 평균을 출력하시오
  • 평균이 90점 이상이면 A, 80점 이상이면 B, 70점 이상이면 C, 60점 이상이면 D
  • 단 평균에 따른 학점산출은 switch문을 사용하여 작성할것
