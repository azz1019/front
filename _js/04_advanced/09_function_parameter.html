상위폴더로 돌아가기

자바스크립트의 파라미터 특징과기능

  • 파라미터의 개수와 관계없이 함수나 메서드의 이름이 같다면 호출이가능
  • 파라미터의 개수가 불일치하여 값을받지않는 파라미터는 undefined처리
  • 값을 받아오지 않는 파라미터가 undefined일때 별도로 파라미터에 기본값을 설정할수있다 => Optional Parameter
  • 모든 파라미터를 관리하는 arguments라는 프로퍼티를 이용해여 파라미터를 배열에담아 사용할수도있다, 프로토타입의 속성(프로퍼티)을 사용하는방식, Optional Parameter의 값은 저장하지 않는다!
