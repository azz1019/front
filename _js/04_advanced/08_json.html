JSON : JavaScript Object Notation : 자바스크립트 객체 표기법

  • 데이터의 요청 및 응답처리에 필요한 데이터의 표준형식을 제공하는 타입
  • 백엔드 언의 종류에 관계없이 데이터 송수신을 단일 형식으로 통일함
  • 데이터의 전송은 모두 string타입으로 이루어지기 때문에 송수신 모듈에서 타입의 변환이 필요함
  • 데이터송신측 : json to string, JSON.stringify(js object)
  • 데이터수신측 : string to json, JSON.parse(string data)
