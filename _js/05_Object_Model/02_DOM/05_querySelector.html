상위폴더로 돌아가기

querySelector(), querySelectorAll() : HTML노드(관리객체)로 리턴하는 메서드

  1. document.querySelector('css selector') : 단일노드로 리턴, 엘리먼트와 비슷
  2. document.querySelectorAll('css selector') : 노드리스트로 리턴, HTMLCollection과 비슷
  3. css selector를 사용하기 때문에 getElement계열 메서드보다 더 선호함
  4. 단, 리턴의 형태가 조금차이가 있기때문에 이 메서드만 사용하는것은 효율적이지 않다
