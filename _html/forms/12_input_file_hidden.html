파일타입은 일반 input과 속성이 다름

파일첨부 윈도우(팝업창)은 html에서 제공하는 고유기능(수정불가)

파일이 첨부된 form은 enctype의 값이 multipart/form-data를 지정해야함

파일전송은 반드시 post방식으로 전송해야한다(전송데이터 크기의한계)




file type은 보안정책상 수정관련 기능을 제공하지 않는다

value속성값에 기존 정보를 로딩할 수 없다

hidden타입은 화면에 출력되지 않은 상태에서 데이터 전송이 필요할때 사용-많은 부분에서 사용이 되므로 꼭 알아둘것!

데이터 전송 목적이므로 name속성과 value속성이 반드시 명시되어야함
